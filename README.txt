Initial Setup Entri Data Siswa Program
1. Create PostgreSQL with the name "biodata_karyawan" (without quot)
2. Import SQL file named "biodata_karyawan.sql" in root project
3. Create ODBC for connect to Power Builder
4. Create ODBC Profile in Power Builder & Replace Database Connection Syntax to Custom class "nvo_apps" in method of_logon()
5. Run Project
